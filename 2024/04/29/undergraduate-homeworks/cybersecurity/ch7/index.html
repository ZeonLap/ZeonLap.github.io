Ch7 TCP/IP 协议栈安全

1. 请描述 IP 分片污染攻击的原理与攻击者需要具备的能力。

  • IP 地址伪造能力，因为攻击者需要冒充成源主机发送伪造的 IP 分片。
  • 需要猜出源主机分配的 IPID。
  • 原始报文校验和的欺骗能力。

2. 结合几个针对 DNS 域名服务实施的 DDoS 攻击案例分析提升 DDoS 攻击防御能力的可行措施

针对 DNS 域名服务实施的 DDoS 攻击需要攻击者操纵僵尸网络伪造许多 SYN 请求，从而维护许多 TCP 半连接，消耗域名服务器的资源，最后资源被耗尽，域名服务器无法工作。措施有：

  • 在收到 TCP SYN 并返回 TCP SYN-ACK 包时，不分配一个专门的数据区，而是根据 SYN 包算出一个 Cookie，这个 Cookie 作为将要返回的 SYN-ACK 包的初始序列号。当客户端再次返回一个 ACK 包时，根据包头信息重新计算 Cookie，然后与返回的确认序列号进行对比，如果相同，那么是一个正常连接，这时才分配资源。
  • AWS 在 2020 年 2 月缓解了一次大规模 DDoS 攻击，这次攻击持续了三天，峰值流量速率达到 2.3 Tb/s。AWS 采取的其中一种应对措施是负载均衡。这样多余的流量会被分配到不同的节点。
  • 另外也可以从应用层解决：可以使用 Web 应用程序防火墙 WAF 检测和过滤恶意 Web 请求。
