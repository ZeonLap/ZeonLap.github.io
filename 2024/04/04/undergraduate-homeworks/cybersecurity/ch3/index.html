第3讲 数据加密

公钥密码的出现解决了对称加密算法的什么问题？但对称加密至今仍被广泛使用，请至少从两个角度简述对称加密算法未被淘汰的原因

解决了对称加密算法的密钥分发问题。在密钥的分发过程中，可能存在密钥窃听者。另外，数字签名也需要公钥密码。

对称加密算法至今未被淘汰的原因：

  • 对称加密算法计算速度更快，在计算效率上对称加密更优秀
  • 公钥密码并不比对称加密算法更安全，公钥密码容易受到暴力破解攻击

散列函数和加密算法同样通过多轮数学计算得出，请至少指出两点原因，说明散列函数不可被加密算法替代

  • 散列函数生成的 Hash Code 往往是定长而且比较短的，计算时间也更短。如果用加密算法代替散列函数来验证消息，会导致存储空间和计算时间的浪费。
  • 在某些情境下，我们只希望验证消息的完整性而不希望接收方能完整地复原整个消息。这时候加密算法就满足不了条件。
