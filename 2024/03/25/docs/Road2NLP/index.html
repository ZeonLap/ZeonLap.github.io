LLaVA

GPT-assisted Visual Instruction Data Generation

  • Background
    • We have image-text dataset like CC, LAION. But they are not good enough
  • Solution
    • Use GPT-4 (text only models) to boost the text dataset
    • Note that the visual image is not used to prompt GPT
  • How to prompt?
    • Captions & Bounding boxed + few human designed examples
  • What to collect?
    1. VQA (conversation)
    2. Describe the image in detail
    3. In-depth reasoning based on 1 and 2

Visual Instruction Tuning

Language Model: LLaMA

Vision Enc: CLIP ViT-L/14

Projection: linear w/o bias

Dataset Preparation

第一轮：问题在前 or 图片在前是随机的

只在绿色的 Token 上计算 Loss (为什么X_instruct 后的 也要？)

Training Scheme

  1. Pre-training for Feature Alignment

    Learn Projection $\textbf{W}$ only

    只使用一轮的对话，而且只预测 Caption

    $\mathbf{X_a}$​ 的 Ground Truth 是原先的 Caption

  2. Fine-tuning End-to-End

    Learn Projection $\mathbf{W}$ + LLM (w/o Visual Encoder)

    Train for:

    • Multimodal Chatbot
    • Science QA
